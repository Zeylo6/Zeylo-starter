Zeylo – Peer-to-Peer Experience Discovery App

Zeylo is a mobile application designed to help people discover and book authentic local experiences hosted by individuals. The idea behind the platform is simple: make it easier for users to explore meaningful activities while supporting small-scale local hosts.

This project was developed as part of the Software Development Group Project (SDGP) module for the BSc (Hons) Computer Science program at IIT in collaboration with the University of Westminster.

Project Idea

Most experience platforms focus on commercial tours or large providers. Zeylo aims to create a more personal and community-driven space where:

Hosts can share unique local experiences

Users can discover activities based on mood, category, or location

Payments are handled securely after the experience is completed

The goal is to reduce risk, increase trust, and promote authentic social interaction.

Main Features
User Management

User registration and login

Support for both hosts and seekers

Profile management

Experience Discovery

Search and filter by category, date, price, and location

Interactive map view

Mood-based filtering

AI-assisted search suggestions

Booking System

Select date and number of guests

Real-time availability checking

QR code for booking confirmation

Payment System

Secure payment gateway integration

Payment processed after the experience

No card details stored in the system

Ratings and Reviews

Hosts and users can rate each other

Helps maintain quality and trust
